## Concept

- 핵심 테마: crew coordination + tmux pane multiplexing
- 아이콘: pane grid와 helm을 결합한 조정 심볼
- 마스코트: 여러 작업 패널을 관리하는 captain 캐릭터

## Usage Notes

- 아이콘은 앱 아이콘, README 배지, 소셜 프로필용 정사각 자산으로 사용
- 마스코트는 README hero, docs, website, release note artwork에 사용
- hero / social 배너는 PNG를 바로 써도 되고, 필요하면 SVG 원본을 수정해 재생성
- 정사각 자산은 PNG로 바로 사용 가능
- 다크 UI에서는 아이콘 원본을 그대로 쓰고, 밝은 배경에서는 16px 이상의 여백을 두는 편이 안정적임
